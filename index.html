websmith.js

A minimalistic static website generator based on Node.js

  • Source code: https://github.com/mbruno46/websmith.js/
  • Bug reports https://github.com/mbruno46/websmith.js/issues

Installation

The package can be downloaded from GitHub using

git clone https://github.com/mbruno46/websmith.js.git

To generate this documentation check the examples directory and type 

cd websmith.js
node index.js ./examples/config.json

Warning!

Node.js must be installed in the system.
